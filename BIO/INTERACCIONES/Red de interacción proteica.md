Conjunto de todas las [[Interacción proteína-proteína|interacciones entre proteínas]] en un sistema biológico.

## El interactoma

La red completa de interacciones de un organismo se llama **interactoma**. Representa:
- Todas las [[Proteína]]s como nodos
- Todas las interacciones como aristas (conexiones)
- La "maquinaria molecular" del organismo

## Propiedades de las redes

Las redes de interacción tienen características típicas:
- **Scale-free**: algunos nodos (hubs) tienen muchas conexiones
- **Modular**: grupos de proteínas interconectadas (módulos funcionales)
- **Robusta**: tolerante a fallos aleatorios pero sensible a hubs

## Bases de datos

Recursos importantes:
- **STRING**: base de datos de interacciones conocidas y predichas
- **BioGRID**: repositorio de interacciones experimentales
- **IntAct**: datos curados de interacciones

## En modelos computacionales

Los [[Modelo de lenguaje de proteínas]] pueden predecir nuevas interacciones para expandir redes conocidas. Esto permite:
- Descubrir funciones de proteínas desconocidas
- Identificar posibles dianas terapéuticas
- Entender enfermedades como perturbaciones de la red
